
Maseru Community Media
Maseru Community Media is a local media platform focused on sharing news, community stories, events, and videos from Maseru, Lesotho. This website aims to inform, educate, and empower the community through engaging content.

About
Maseru Community Media provides a platform where residents and visitors can stay updated with:
Local news and community stories
Upcoming events in Maseru
Interviews and videos from youth and other community members
Opportunities to engage and contact the team
The website is designed to be simple, informative, and user-friendly, providing all essential information in an organized manner.
Features
Responsive layout for mobile and desktop
Navigation bar for easy access to all pages
Image and video integration
Informative sections including News, Events, Videos, and Contact
Pages
Home (index.html) – Introduction to the website and its mission.
News (news.html) – Latest news and community stories.
Events (events.html) – List of upcoming events in Maseru.
Videos (videos.html) – Embedded videos featuring community voices.
Contact (contact.html) – Contact information for submissions and inquiries.
Technologies Used
HTML5
CSS (linked via style.css)
Embedded YouTube videos
Responsive design principles
How to View
 download the repository.
Open the index.html file in your preferred web browser.
Navigate through the website using the top navigation menu.
